2. 研究背景

深度生成模型是机器学习领域的核心挑战之一，旨在从高维空间中学习复杂的数据分布。目前主流的生成模型包括：

  • 生成对抗网络（GAN）：通过生成器与判别器的对抗训练生成高质量样本，但对数似然难以计算或不佳。
  • 流模型（Flow-based）：通过可逆变换建模数据分布，计算成本高。
  • 自回归模型（如PixelCNN）：逐像素建模，生成质量好但速度慢。
  • 变分自编码器（VAE）：基于变分推断和重参数化技巧优化对数似然的证据下界（ELBO）。

VAE的ELBO目标函数为：

\[ \ln p_ heta(x) \geq \mathbb{E}_{q_\phi(z|x)}[\ln p_ heta(x|z) + \ln p_ heta(z) - \ln q_\phi(z|x)] riangleq \mathcal{L}(x; heta,\phi) \]

传统VAE使用标准高斯先验 \(p_ heta(z) = \mathcal{N}(0,I)\)。但近期研究[26]表明，这种简单先验会导致欠拟合。因此，许多可学习先验被提出，大多聚焦于逼近聚合后验 \(q_\phi(z)\)——理论上最大化ELBO的最优先验。然而，现有方法的性能有限，且聚合后验的实际意义不明确。

本文注意到判别器可以评估数据的质量，并认为应当利用判别器来调整可学习先验，其中判别器具有明确的实际意义。

3. 论文目的与解决的问题

本文旨在解决以下关键问题：

  1. 先验分布过于简单：标准高斯先验无法捕捉复杂数据分布，导致欠拟合。
  2. 聚合后验先验的局限性：现有基于聚合后验的可学习先验性能有限，且缺乏实际解释。
  3. VAE采样质量差：相比GAN，VAE生成的样本通常更模糊，缺乏锐利细节。
  4. 训练稳定性：VAE与GAN的混合训练往往不稳定。

为此，本文提出Pull-back Prior和VAEPP框架，通过判别器评估数据质量来调整先验密度，从而同时提升对数似然和采样质量。

4. 核心原理

4.1 Pull-back Prior定义

\[ p_\lambda(z) = rac{1}{Z} p_N(z) \cdot e^{-eta D(G(z))} \]

其中 \(p_N\) 是简单先验（如标准高斯），\(D\) 是判别器，\(G\) 是生成器（取 \(p_ heta(x|z)\) 的均值），\(eta\) 是可学习标量，\(Z\) 是配分函数。

4.2 最优\(eta\)的梯度推导

\[ rac{\partial \mathcal{L}}{\partial eta} = -\mathbb{E}_{q_\phi(z)}[D(G(z))] + \mathbb{E}_{p_\lambda(z)}[D(G(z))] \]

当训练收敛时，\(\partial\mathcal{L}/\partialeta = 0\)，意味着判别器无法区分重构数据和生成数据，与GAN的哲学一致。

4.3 ELBO中的可学习先验项

\[ \mathcal{L}( heta,\phi,\lambda) = \mathbb{E}_{p^*(x)}\mathbb{E}_{q_\phi(z|x)}[\ln p_ heta(x|z)] + \mathbb{E}_{p^*(x)}H[q_\phi(z|x)] + \mathbb{E}_{q_\phi(z)}[\ln p_\lambda(z)] \]

4.4 Wasserstein距离

\[ W_1(\mu, u) = \sup_{ ext{Lip}(D)\leq 1}\{\mathbb{E}_{\mu(x)}D(x) - \mathbb{E}_{ u(x)}D(x)\} \]

4.5 配分函数上界

\[ \hat{Z} = \mathbb{E}_{p^*(x)}\mathbb{E}_{q_\phi(z|x)}\left[ rac{f_\lambda(z)}{ rac{1}{N}q_\phi(z|x)} ight] \geq \mathbb{E}_{q_\phi(z)}\left[ rac{f_\lambda(z)}{q_\phi(z)} ight] = Z \]

4.6 Pull-back Prior的理论推导

通过变分法求解约束优化问题：

\[ \min_\lambda\{\mathbb{E}_{p_\lambda(z)}[D(G(z))] - \mathbb{E}_{p^*(x)}[D(x)]\} \quad ext{s.t.} \quad ext{KL}(p_\lambda, p_N) = lpha, \int p_\lambda(z)dz = 1 \]

利用欧拉-拉格朗日方程，得到最优解为Pull-back Prior的形式。

6.1 1步训练算法（VAEPP）

算法2: VAEPP 1步训练算法

输入: 梯度惩罚方法R, batch大小b, Adam参数τ
输出: 训练好的θ, φ, β, ω

1: while θ, φ, β, ω 未收敛 do
2:    for i = 1 to b do
3:        采样 e, x ~ p*, z ~ q_φ(z|e), ε ~ p_N
4:        计算 ê ← G(ε), 梯度惩罚 ζ ← R(e, ê)
5:        计算 Z^(i) = 0.5 * (e^{-βD(G(ε))} + f_λ(z) / (M * q_φ(z|e)))
6:        计算 L^(i) = ln p_θ(x|z) + ln f_λ(z) - ln q_φ(z|e) + βζ
7:    end for
8:    θ, φ, β, ω ← Adam(∇(1/b ΣL^(i) - ln(1/b ΣZ^(i))), {θ,φ,β,ω}, τ)
9: end while

算法1: Naive VAEPP 2步训练算法

输入: 梯度惩罚算法R, batch大小b, critic迭代数n_c, Adam参数τ
输出: 训练好的θ, φ, β, ω

1: while θ, φ, β, ω 未收敛 do
2:    for k = 1 to n_c do
3:        // VAE步骤：更新θ, φ, β
4:        采样并计算L^(i)和Z^(i)
5:        θ, φ, β ← Adam(∇(1/b ΣL^(i) - ln(1/b ΣZ^(i))), {θ,φ,β}, τ)
6:    end for
7:    // WGAN步骤：更新ω（判别器）
8:    采样 e, x ~ p*, z ~ p_N, ê ← G(z)
9:    计算梯度惩罚 ζ ← R(e, ê)
10:   L_D ← D(x̂) - D(x) + ζ
11:   ω ← Adam(∇(1/b ΣL_D), ω, τ)
12: end while

7. 实现细节

编码器：使用截断正态分布（丢弃偏离均值超过2个标准差的样本）代替正态分布，显著降低Ẑ的方差（MNIST上从0.809260降至0.000967）。

解码器：MNIST上使用Bernoulli分布，CIFAR-10上使用Discretized Logistic分布。

生成器G(z)：取解码器均值 \(G(z) = \mathbb{E}_{p_ heta(x|z)}[x]\)。

梯度惩罚策略：默认使用WGAN-div-1（在真实数据和生成数据的线性插值上采样），在三类策略（WGAN-GP, WGAN-div-1, WGAN-div-2）中表现最佳。

采样过程：三步采样——(1) 用GAN从 \(q_\phi(z)\) 采样初始 \(z_0\)；(2) 通过Metropolis-Adjusted Langevin Algorithm (MALA)从 \(p_\lambda(z)\) 采样；(3) 用解码器生成图像。

隐空间维度选择：在CIFAR-10上，隐维度d_z=128时获得最佳FID。隐维度增大时，BPD持续改善但采样质量下降。

11. 工程落地与部署

graph LR subgraph VAEPP部署 A[训练数据] --> B[编码器 q_φ] B --> C[隐变量 z] C --> D[解码器 p_θ] D --> E[生成图像] C --> F[Pull-back先验评估] F --> G[判别器 D] G --> H[质量评分] H --> I[朗之万采样优化] I --> E end

VAEPP的训练和评估较为耗时（因额外判别器和朗之万动力学），暂未应用自回归组件。隐空间维度d_z应选略大于真实数据流形维度的数值。在CIFAR-10上，d_z=128时获得最佳FID。使用截断正态分布可显著降低估计方差。

12. 局限性与未来工作

  • 自回归组件缺失：由于训练和采样已很耗时，VAEPP未集成自回归组件（如PixelCNN），若能集成可能进一步提升性能。
  • 隐空间维度选择：FID/IS与BPD的变化趋势不一致，需要针对不同指标分别调优隐维度。
  • 配分函数估计：当前使用重要性采样估计Z的上界，估计方差虽可接受但仍有改进空间。
  • 大规模数据集：当前仅在MNIST、CIFAR-10、CelebA等中等规模数据集上验证，ImageNet等更大规模数据集上的性能有待验证。
  • 朗之万动力学效率：采样过程需要多步迭代，生成速度受限。

13. 总结与启示

VAEPP通过Pull-back Prior这一新颖的可学习先验，成功弥合了VAE与GAN之间的差距：

  1. 判别器指导先验学习：利用判别器评估数据质量的能力来调整隐空间先验密度，使先验具有明确的物理意义。
  2. 1步训练框架：将VAE和WGAN的优化合并为统一的端到端训练，提高了训练稳定性和效率。
  3. 显式先验的优势：相比2-Stage VAE的隐式先验，Pull-back Prior的显式形式使得朗之万动力学采样可行，进一步提升了采样质量。
  4. SOTA性能：在无需自回归组件的条件下，VAEPP的对数似然达到最优，采样质量接近GAN。

2a. 研究背景深入分析

2a.1 VAE的发展与挑战

变分自编码器自2013年提出以来，经历了多次重要改进。原始VAE[10,21]使用标准高斯先验和简单的编码器-解码器结构。Kingma等人[12]提出逆自回归流（Inverse Autoregressive Flow, IAF），通过丰富变分后验来提升ELBO。Tomczak和Welling[26]提出VampPrior，使用混合分布作为可学习先验，逼近聚合后验。Bauer和Mnih[2]提出重采样先验，通过重要性采样从简单先验构造更丰富的先验。

尽管这些方法在丰富后验或先验方面取得了进展，但仍存在根本性问题：基于聚合后验的方法[2,25,26]性能有限，且聚合后验的物理意义不清晰。VAEPP通过引入判别器评估数据质量来调整先验，提供了更直观的先验设计思路。

2a.2 GAN的发展与Wasserstein距离

生成对抗网络[6]通过生成器与判别器的对抗训练实现高质量样本生成。然而，原始GAN训练不稳定，容易模式崩溃。WGAN[1]引入Wasserstein距离替代JS散度，显著提升了训练稳定性。WGAN-GP[7]通过梯度惩罚代替权重裁剪，进一步改善了Lipschitz约束的实现。WGAN-div[28]提出了更灵活的散度度量。

VAEPP利用WGAN的梯度惩罚技术，但目的不同——不是为了训练独立的GAN，而是将判别器融入VAE的训练过程，用于调整先验分布。

2a.3 朗之万动力学与MCMC采样

朗之万动力学（Langevin Dynamics）是一种基于随机微分方程的MCMC采样方法。给定目标分布p(z)，采样迭代公式为：

\[ z_{t+1} = z_t + rac{\epsilon}{2} abla_z \log p(z_t) + \sqrt{\epsilon}\eta_t, \quad \eta_t \sim \mathcal{N}(0, I) \]

Metropolis-Adjusted Langevin Algorithm (MALA)在朗之万动力学基础上添加了接受-拒绝步骤，保证采样的正确性。Kumar等人[13]在能量模型中使用MALA进行采样，其中密度函数也包含判别器项。VAEPP借鉴了这一方法，从显式先验p_λ(z)中采样。

2a.4 VAE与GAN的融合探索

VAE和GAN各有优劣：VAE有明确的对数似然目标但采样质量差，GAN采样质量好但缺乏似然保证。许多工作尝试融合两者：

  • VAE-GAN[18]：使用GAN的判别器作为VAE的额外损失项，改善重构质量。
  • 2-Stage VAE[4]：先训练标准VAE，再用GAN学习隐空间的先验分布。
  • WAE[18]：使用Wasserstein距离替代KL散度作为正则项。

VAEPP与这些方法的根本区别在于：判别器不是用于匹配隐空间分布或改善重构，而是用于调整先验密度——增加生成好数据的z的密度，降低生成差数据的z的密度。

4a. 核心原理深入推导

4a.1 Pull-back Prior的变分推导

Pull-back Prior可以通过变分法严格推导。考虑以下约束优化问题：在给定θ, φ的条件下，搜索使W¹(p_θ, p*)最小化的先验p_λ：

\[ \min_\lambda \sup_{ ext{Lip}(D) \leq 1} \{\mathbb{E}_{p_\lambda(z)}\mathbb{E}_{p_ heta(x|z)}[D(x)] - \mathbb{E}_{p^*(x)}[D(x)]\} \]

引入假设 \(\mathbb{E}_{p_ heta(x|z)}[D(x)] = D(G(z))\)，并使用近似的判别器\( ilde{D}\)代替D，简化后的优化问题为：

\[ \min_\lambda \{\mathbb{E}_{p_\lambda(z)}[D(G(z))] - \mathbb{E}_{p^*(x)}[D(x)]\} \quad ext{s.t.} \quad ext{KL}(p_\lambda, p_N) = lpha \]

利用拉格朗日乘子法和欧拉-拉格朗日方程，得到最优解为Pull-back Prior的形式，其中β=1/γ（γ为KL约束的拉格朗日乘子），ln Z = (1 + η/γ)（η为归一化约束的拉格朗日乘子）。

4a.2 1步训练与2步训练的等价性

定理3证明了1步训练的优化过程等价于最大化K项（ELBO中与先验相关的项），且K是βW¹(p†, p*)的下界：

\[ K = -\mathbb{E}_{q_\phi(z)}[eta \cdot D(G(z))] - \ln Z \leq eta \max_{ ext{Lip}(D)\leq1}\{\mathbb{E}_{p^\dagger(x)}[D(x)] - \mathbb{E}_{p^r(x)}[D(x)]\} = eta W^1(p^\dagger, p^r) \]

其中p†(x) = E_{p_N(z)}[p_θ(x|z)]为从简单先验生成的数据分布，p^r(x) = E_{q_φ(z)}[p_θ(x|z)]为从编码器采样的重构数据分布。实验观察到p^r ≈ p*。

4a.3 配分函数Z的估计策略

配分函数Z的精确计算在高维空间中是不可行的。VAEPP使用重要性采样估计Z的上界Ż。关键选择是建议分布——使用q_φ(z)还是p_N(z)：

  • 当KL(p_N, p_λ)较小时（训练初期），p_N(z)作为建议分布是合理的。
  • 当KL(q_φ, p_λ)远小于KL(p_N, p_λ)时（训练后期），q_φ(z)更优。

实验表明，即使在MNIST上，KL(q_φ, p_λ) = 12.497，而KL(p_N, p_λ) = 57.45，差距近5倍。在CIFAR-10上差距更大（180.3 vs 1011.30）。因此VAEPP默认使用q_φ(z)作为建议分布。

4a.4 Bernoulli数据集的特殊处理

对于MNIST等二值图像数据集，通常从灰度图像（连续值）采样二值图像。这导致样本空间指数级增大（2^像素数），直接使用公式(7)估计Ż会严重高估Z。VAEPP提出使用p*(e)（灰度图像分布）代替p*(x)（二值图像分布）来估计q_φ(z)的下界：

\[ q_\phi(z) = \mathbb{E}_{p^*(x)}[q_\phi(z|x)] = \mathbb{E}_{p^*(e)}[q_\phi(z|e)] \]

其中q_φ(z|e) = E_{p*(x|e)}[q_φ(z|x)]，即从灰度图像e的条件分布中采样二值图像x，再计算q_φ(z|x)的期望。这种方法显著降低了Ż的方差。

7a. 实现细节深入

7a.1 截断正态分布的实现

编码器q_φ(z|e)使用截断正态分布，丢弃偏离均值超过2个标准差的样本。这一简单改动对Ż的方差影响巨大：

  • 标准正态分布：Var(Ż) = 0.809260（不可接受）
  • 截断正态分布：Var(Ż) = 0.000967（可接受，10⁹样本）

截断正态分布减少了q_φ(z|x)和(1/M)q_φ(z|x)之间的差距，因为截断使分布更集中，单个样本的q_φ(z|x)值更接近(1/M)q_φ(z)。

7a.2 可学习方差γ的设计

解码器p_θ(x|z)的方差设为可学习标量γ。这一设计受Dai和Wipf[4]的启发：当隐维度d_z增大时，FID和IS的变化趋势与BPD相反。通过将方差设为可学习参数，模型能够自适应地平衡重构精度和采样质量。

实验发现，VAEPP在d_z=128时达到最佳FID（CIFAR-10上71.0），而BPD在d_z=1024时仍然在改善。这说明d_z应选择略大于真实数据流形维度的数值。

7a.3 1步训练vs 2步训练的稳定性

2步训练（Naive VAEPP）的主要问题是不稳定性：两个独立优化（VAE和WGAN）交替进行，可能导致训练损失震荡甚至崩溃。在CIFAR-10上，Naive VAEPP在约80个epoch时几乎崩溃。

1步训练（VAEPP）将两个优化合并为一个，梯度惩罚项乘以β后加入ELBO损失。由于β在训练过程中自适应调整，梯度惩罚的影响也会相应变化，从而保持训练稳定性。

7a.4 IvOM指标

IvOM（Importance weighted Oracles Metric）[19]是评估生成模型数据覆盖率的指标。VAEPP在MNIST和CIFAR-10上分别达到0.018和0.017的IvOM值，表明模型具有良好的数据覆盖能力。

7a.5 朗之万动力学的初始点选择

朗之万动力学要求初始点z₀的密度p_λ(z₀)足够高，否则采样链需要很长时间才能收敛。VAEPP使用GAN从q_φ(z)中采样初始z₀，因为q_φ(z)与p_λ(z)接近（KL散度小），保证初始密度足够高。

实验比较了从z₀直接解码（GAN-VAEPP）和经过朗之万动力学后再解码（VAEPP）的采样质量。在MNIST、Fashion-MNIST和CIFAR-10上，VAEPP的FID通常优于GAN-VAEPP，验证了朗之万动力学对采样质量的提升作用。

8a. 实验结果深入分析

8a.1 隐维度对多指标的影响

隐维度d_z的选择对BPD、FID和IS三个指标有截然不同的影响：

  • BPD：随d_z增大持续改善（更多参数拟合数据）。
  • FID：在d_z=128时达到最优，之后开始恶化（过多参数导致采样质量下降）。
  • IS：在d_z≈128时也达到最优。

这一现象验证了Dai和Wipf[4]的命题：d_z应选择为活跃隐维度的最小数量。当d_z过大时，冗余维度引入噪声，降低了采样质量。

8a.2 对抗验证损失分析

FGSM攻击下，不同稀疏度模型的对抗验证损失曲线显示：

  • 稀疏度P_m=16.9%时，模型在验证集上达到最低对抗损失，说明这是MNIST Digits上的"最佳稀疏度"。
  • 继续增加稀疏度（P_m=3.6%, 1.8%），对抗损失开始上升，但即使在P_m=1.8%时，对抗准确率仍保持在90%以上。

8a.3 初始权重分布分析

对中奖彩票在不同稀疏度下的初始权重分布进行分析，发现：

  • 随着剪枝进行，保留的权重趋向于具有更大的初始绝对值。
  • FGSM对抗训练和PGD对抗训练下的权重分布与正常训练的分布有所不同，表明对抗训练确实选择了不同的子网络结构。
  • 第一隐藏层的权重分布最为集中，第二隐藏层次之，输出层最分散。

14a.2 关键代码片段

# VAEPP 1步训练伪代码
def vaepp_train_step(batch, encoder, decoder, discriminator, gp_method, beta, optimizer):
    e, x = batch  # 灰度图像和二值图像
    z = encoder.sample(e)  # 从q_φ(z|e)采样
    eps = torch.randn_like(z)  # 从p_N采样
    e_hat = decoder.mean(eps)  # G(ε)
    zeta = gp_method(e, e_hat)  # 梯度惩罚项
    
    # 估计Ż
    f_lambda_z = prior_N(z) * torch.exp(-beta * discriminator(e_hat))
    Z_hat = 0.5 * (torch.exp(-beta * discriminator(e_hat)) + 
                    f_lambda_z / (M * encoder.prob(z, e)))
    
    # 计算ELBO
    recon_loss = decoder.log_prob(x, z)
    prior_loss = torch.log(f_lambda_z)
    enc_loss = encoder.log_prob(z, e)
    L = recon_loss + prior_loss - enc_loss + beta * zeta
    
    # 梯度更新
    loss = torch.mean(L) - torch.log(torch.mean(Z_hat))
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    return loss.item()

附加A. 理论深度分析

A.1 ELBO的三项分解

ELBO可以分解为三项：

\[ \mathcal{L}(\theta, \phi, \lambda) = I + J + K \]

其中：

  • I = E_{p*(x)} E_{q_φ(z|x)}[ln p_θ(x|z)]：重构项
  • J = E_{p*(x)} H[q_φ(z|x)]：后验熵项
  • K = E_{q_φ(z)}[ln p_λ(z)]：先验项

注意p_λ(z)只出现在K项中。最优先验是q_φ(z)（聚合后验），但实际中难以直接使用。

A.2 Pull-back Prior与GAN哲学的统一

当训练收敛时，∂L/∂β = 0意味着：

\[ \mathbb{E}_{q_\phi(z)}[D(G(z))] = \mathbb{E}_{p_\lambda(z)}[D(G(z))] \]

这表明判别器无法区分重构数据（来自q_φ(z)）和生成数据（来自p_λ(z)）。这与GAN的纳什均衡条件一致，但形式不同：GAN直接在数据空间训练，VAEPP在隐空间训练。

A.3 Wasserstein距离与W¹的关系

VAEPP使用1-Wasserstein距离（W¹）作为分布差异度量。W¹具有以下性质：

  • 对称性：W¹(μ, ν) = W¹(ν, μ)
  • 三角不等式：W¹(μ, ξ) ≤ W¹(μ, ν) + W¹(ν, ξ)
  • 对支撑集不重叠的分布也有良好定义（不像KL散度）

W¹的对偶形式（Kantorovich-Rubinstein对偶）将无限维优化问题转化为1-Lipschitz函数空间的优化，这正是GAN中判别器D的角色。

A.4 朗之万动力学的收敛性

朗之万动力学的稳态分布等于目标分布p_λ(z)。在以下条件下保证收敛：

  • 步长ε足够小
  • ∇ln p_λ(z)是Lipschitz连续的
  • p_λ(z)满足log-Sobolev不等式

VAEPP中使用MALA（Metropolis-Adjusted Langevin Algorithm），通过接受-拒绝步骤保证采样的精确性。

附加B. 实现细节深入

B.1 编码器架构

编码器q_φ(z|x)使用截断正态分布：

\[ q_\phi(z|x) = \mathcal{TN}(\mu_\phi(x), \sigma_\phi^2(x), -2\sigma_\phi, 2\sigma_\phi) \]

截断区间为[-2σ, 2σ]，覆盖约95%的标准正态分布。这一设计的好处：

  • 避免极端样本对Ẑ估计的影响
  • 减少q_φ(z|x)和(1/M)q_φ(z|x)的差距
  • 提高估计稳定性

B.2 解码器选择

数据集 解码器类型 方差

Static-MNIST Bernoulli 固定
Fashion-MNIST Discretized Logistic 可学习γ
CIFAR-10 Discretized Logistic 可学习γ

B.3 判别器架构

判别器D使用CNN结构（针对图像数据），类似于WGAN的判别器。关键设计：

  • 无BatchNorm（与Lipschitz约束兼容性问题）
  • 使用LeakyReLU激活
  • 最后输出单一标量（无sigmoid）
  • 梯度惩罚保证近似1-Lipschitz

B.4 训练优化器

所有参数使用Adam优化器，超参数：

  • 学习率：初始1e-4，每200 epoch减半
  • β1 = 0.5（与GAN训练常用值一致）
  • β2 = 0.9
  • 梯度裁剪：[-10, 10]

附加C. 采样过程详解

C.1 三步采样流程

  1. 初始点生成：用预训练的GAN（建模q_φ(z)）采样z_0
  2. 朗之万动力学优化：从z_0开始，通过MALA迭代获得z ~ p_λ(z)
  3. 图像生成：x = decoder.mean(z)

C.2 GAN-VAEPP vs VAEPP

GAN-VAEPP直接从z_0生成图像，跳过朗之万动力学步骤。比较两者：

方法 采样步骤 FID（CIFAR-10） 速度
GAN-VAEPP z_0直接解码 74.1 快
VAEPP z_0 → MALA → 解码 71.0 慢（MALA迭代）

MALA采样可以"修正"z_0使其更符合p_λ(z)的高密度区域，提升采样质量。

C.3 拒绝采样的局限

Accept-Reject采样[2]需要p_λ(z)/p_N(z)在p_λ支撑集上有上界T。VAEPP中难以保证存在小的T，因为：

  • 判别器D(G(z))的范围可能很大
  • β的可学习性使得比值动态变化
  • 支撑集理论上是整个隐空间，难以约束

因此VAEPP采用MALA而非拒绝采样。

C.4 多模式采样

朗之万动力学的一个挑战是多模态分布的采样：链可能陷入局部模式。VAEPP通过以下方式缓解：

  • 使用GAN生成多样化的初始点z_0
  • 设置较大的步长ε，鼓励探索
  • 多链并行采样

附加D. 评估指标详解

D.1 NLL（Negative Log-Likelihood）

NLL衡量模型对测试数据的拟合程度：

\[ \text{NLL} = -\mathbb{E}_{x \sim p^*}[\log p_\theta(x)] \]

计算p_θ(x)使用重要性采样：

\[ \log p_\theta(x) \approx \log \frac{1}{K}\sum_{k=1}^K \frac{p_\theta(x, z_k)}{q_\phi(z_k|x)}, \quad z_k \sim q_\phi(z|x) \]

K越大估计越精确，但计算成本越高。VAEPP使用10⁸-10⁹个样本进行评估。

D.2 Bits per Dimension (BPD)

BPD是连续值数据集（如CIFAR-10）的常用指标：

\[ \text{BPD} = \frac{-\log p_\theta(x|z)}{D \cdot \ln 2} \]

其中D是数据维度。CIFAR-10为3072维（32×32×3）。

D.3 FID（Fréchet Inception Distance）

FID衡量生成图像与真实图像在Inception网络特征空间的距离：

\[ \text{FID} = \|\mu_r - \mu_g\|^2 + \text{Tr}(\Sigma_r + \Sigma_g - 2\sqrt{\Sigma_r \Sigma_g}) \]

FID越低，生成质量越接近真实数据。

D.4 IS（Inception Score）

IS衡量生成图像的质量和多样性：

\[ \text{IS} = \exp(\mathbb{E}_x[\text{KL}(p(y|x) || p(y))]) \]

IS越高越好。

D.5 IvOM（Importance weighted Oracles Metric）

IvOM衡量生成模型的数据覆盖能力。VAEPP在MNIST上达到0.018，CIFAR-10上达到0.017，表明良好的覆盖能力。

附加G. VAEPP扩展研究方向

G.1 集成自回归组件

VAEPP当前未集成自回归组件（如PixelCNN）。集成方向：

  • 解码器p_θ(x|z)使用PixelCNN结构
  • 挑战：训练和采样开销显著增加
  • 潜在收益：进一步降低NLL

G.2 条件生成

将VAEPP扩展至条件生成：

\[ p_\lambda(z|y) = \frac{1}{Z(y)} p_N(z) \exp(-\beta D(G(z, y), y)) \]

其中y是条件（如类别标签）。这使得VAEPP可以用于条件图像生成、文本到图像等任务。

G.3 半监督学习

VAEPP可扩展至半监督学习：

  • 有标签数据：监督损失 + ELBO
  • 无标签数据：仅ELBO
  • 判别器D：同时评估数据质量和类别匹配

G.4 隐空间几何研究

Pull-back先验提供了对隐空间几何的新视角：

  • 判别器D诱导的隐空间度量
  • 朗之万动力学的轨迹与数据流形的关系
  • 隐插值的语义连续性
