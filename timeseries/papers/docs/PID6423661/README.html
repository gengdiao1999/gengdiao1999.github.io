二、背景（Background）

聚类是无监督方式将相似样本分组。k-means 和 GMM 用简单相似度量和统计分布表示聚类，但需手工设计特征。深度神经网络（DNN）大幅提升聚类精度。

Deep Embedded Clustering (DEC) 同时学习特征和质心，在低维特征上 k-means；但训练目标与经典聚类不完全匹配，效果不佳。

深度生成模型——VAE 和 GAN 的变体。Gaussian Mixture GAN 在隐空间用高斯混合；但 GAN 类对聚类分配和其他隐变量关系约束不强，在 MNIST 之外效果下降。

VAE 路线：Gaussian Mixture VAE (GMVAE) 和 VaDE 是两个典型代表。用 Categorical y + Gaussian z，通过 mean-field 近似 $q(z, y|x) = q(z|x)q(y|x)$ 和 SGVB 训练。但 mean-field 限制 z 与 y 在 x 条件下独立，难以处理真实数据集。

三、目的（Problems Solved）

  • mean-field 限制 z 与 y 依赖：直接用 q(z|y, x)。
  • 类别变量 y 不可重参数化：用 VIMCO 等非重参数化 VI。
  • VIMCO 训练不稳定：std annealing 稳定训练。
  • 类别变量 y 训练困难：NVIL/VIMCO 处理离散 y。
  • 基线算法精度不足：5 个基准超过 GMVAE/VaDE/LTVAE。

四、核心原理（Principles）

系统总览：NVISA 是基于 Gaussian Mixture VAE 的无监督聚类框架。生成 $p_θ(x, y, z) = p_θ(x|z) p_θ(z|y) p_θ(y)$；变分 $q_φ(z, y|x) = q_φ(z|y, x) q_φ(y|x)$。VIMCO 训练 + std annealing 稳定。

关键概念：

  • Gaussian Mixture VAE：Categorical y + Gaussian z。
  • Mean-Field Approximation：$q(z,y|x) = q(z|x)q(y|x)$。
  • VIMCO / NVIL / EnumY：三种非重参数化 VI。
  • Std Annealing：σ 退火。
  • SoftPlus：$\log(e^a+1)$。

数学原理：

  • 生成模型：

\[\]

p_θ(x, y, z) = p_θ(x|z) p_θ(z|y) p_θ(y)


\[\]
  • 变分后验（不用 mean-field）：

\[\]

q_φ(z, y|x) = q_φ(z|y, x) q_φ(y|x)


\[\]
  • VIMCO 目标：

\[\]

L_M(x;θ,φ) = E_{(y,z)^{(1:M)} \sim q_φ(z,y|x)} \left[ \log \frac{1}{M} \sum_{m=1}^{M} \frac{p_θ(x, y^{(m)}, z^{(m)})}{q_φ(z^{(m)}, y^{(m)}|x)} \right]


\[\]
  • NVIL 梯度：

\[\]

\nabla L_{ELBO} = E \nabla f(x,y,z) + (f - C_ψ(x) - c) \nabla \log q_φ(z,y|x)


\[\]
  • Std Annealing：

\[\]

σ'_x = SoftPlus(σ_x) + ϵ_x, \quad σ'_z = SoftPlus(σ_z) + ϵ_z


\[\]

八、关键创新点（Key Innovations）

  • + q(z|y, x) 代替 mean-field：表达更一般的条件依赖。
  • + VIMCO 非重参数化 VI：处理 Categorical y。
  • + Std Annealing：稳定 VIMCO + Gaussian p(x|z)。
  • + 5 个基准数据集 SOTA：MNIST 98.40%、Fashion 66.14%、STL-10 91.22%、CIFAR-10 76.96%、CIFAR-100 38.79%。
  • + AE 预训练 + GMM 初始化：MNIST 进一步 98.04%±0.17%。

九、实验与结果（Experiments）

  • 数据集：MNIST、Fashion-MNIST、STL-10、CIFAR-10、CIFAR-100。
  • Baseline：AE+GMM、VAE+GMM、DEC、GM-GAN、GMVAE、VaDE、LTVAE-GMM、LTVAE-full。
  • 主要指标：Unsupervised Clustering Accuracy (ACC) %。
  • 关键结果数字：MNIST 98.40、Fashion 66.14、STL-10 91.22、CIFAR-10 76.96、CIFAR-100 38.79；NVIL loss 发散；VIMCO 1000 samples 91.3；VIMCO 15 + std anneal 98.40。
  • 消融实验：q(z|y,x) vs mean-field；EnumY vs NVIL vs VIMCO；std annealing 开/关。
  • 效率分析：3-12 小时（CIFAR-100 30 小时），GTX 1080 Ti。
