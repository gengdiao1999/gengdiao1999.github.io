   Abstract—To ensure the reliability of the Internet-based appli-
cation services, KPIs (Key Performance Monitors) are closely
monitored in real time and the anomalies presented in the
KPIs must be discovered in time. While anomaly detection
for the seasonal smooth service-level KPIs (e.g., number of
transactions per minute) have been solved reasonably well in
the literature, the intricate KPIs at the machine level (e.g.,
the number of I/O requests on a server monitored per second)
has been little studied. These intricate KPIs are prevalent and
important, but exhibit non-Gaussian noises and complex data
distribution that are hard to model. In this paper, we propose
an adversarial training method in the Bayesian network based
on partition analysis with solid theoretical proof. Based on it,
we propose the first unsupervised anomaly detection algorithm
Buzz for intricate KPIs with high performance. Its best F-scores
on the data from a global Internet company range from 0.92           Fig. 1: Intricate KPIs in this paper, each of which is plotted
