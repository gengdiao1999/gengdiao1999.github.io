🔷 Shapelet 原语 2009

Time Series Shapelets: A New Primitive for Data Mining

Lexiang Ye, Eamonn Keogh

KDD 2009

时序分类第一篇系统提出"shapelet（最具判别性的子序列原语）"的工作：用信息增益枚举候选子序列，构造可解释决策树 + Shapelet Transform，在 UCR-6 上精度对齐甚至超过 1-NN DTW，并奠定后续 HIVE-COTE、LE Shapelet 等的思路。

📏 距离度量 2018

An Ultra-Fast Time Series Distance Measure to Allow Data Mining in More Complex Real-World Applications

Shaghayegh Gharghabi, Yifei Imani, Anthony Bagnall, Ali Darvishzadeh, Eamonn Keogh

SIAM SDM 2018

基于 Matrix Profile 的 MPdist：用"前 5% 最近子序列距离中位数"的最大对称化定义距离，在保留 DTW 1-NN 精度的同时，m=10⁴ 长序列上比 DTW 快 ~30 倍，并天然支持非等长与自相似序列，是工业级时序距离度量的里程碑。

NeurIPS

面向医疗时序分类的 Transformer：多粒度 patching + 通道内/通道间双向自注意力，在 5 个 EEG/ECG/EMG 公开基准上全面超越 PatchTST、Crossformer 等强基线。
