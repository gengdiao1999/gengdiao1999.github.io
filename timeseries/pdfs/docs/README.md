## 三、关联与同族

- **CN111309565B ↔ CN111338915B**：告警处理双件，前者聚焦"风暴检测+摘要"，后者聚焦"动态定级排序"。

- **CN111737095B ↔ CN113448808B**：批处理双件，前者实时监控，后者单任务时间预测。
- **CN113900844B ↔ CN113962273B**：服务码根因定位姊妹件，后者是前者的"前置异常检测模块"。

- **CN112559237B ↔ CN112559238B**：排障双件，前者通用引擎（抽象排障图），后者 Oracle 场景化。
- **CN110532550A**：日志解析基础，与 CN114818643A（保留业务信息的模板提取）互补。
- **CN110837953A**：异常实体定位，是 CN111444247B / CN111597070B 根因方法的轻量化前身。
