# study

AIOps 个人站点与资料索引仓库。

## 站点目标

这个仓库用于整理 AIOps 学习资料，按三个专题组织：

- `timeseries/`：时间序列
- `logs/`：日志
- `tracing/`：调用链

每个专题统一拆分为四类资料：

- `patents/`：专利索引
- `papers/`：文献索引
- `repos/`：源码与仓库索引
- `notes/`：个人笔记索引

## 维护建议

1. 新增资料时，把原始文件放入对应专题目录。
2. 同步更新该专题下对应分类的 `index.html`。
3. 尽量补齐标题、摘要、标签、适用场景和个人判断。
4. 让专利、文献、源码和笔记之间形成双向链接。
